# 文档修订

本项目已经公开发布到Github，可以通过下面的方式快速定位到相应 Github 页面去编辑当前文档。

- 组内成员可直接编辑。
- 组外成员 fork 后请发起 Pull Request。

- [Markdown 相关知识](https://www.runoob.com/markdown/md-tutorial.html)
- [Vuepress 编辑技巧](https://vuepress.vuejs.org/zh/guide/markdown.htm)
